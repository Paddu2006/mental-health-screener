# Mental Health Screener

A Python tool that administers validated clinical screening tools (PHQ-9, GAD-7, PSS-10) and uses Machine Learning to assess mental health risk.

Built as Phase 3 - Project 5.

## Features
- PHQ-9 Depression Screening (9 questions)
- GAD-7 Anxiety Screening (7 questions)
- PSS-10 Stress Assessment (10 questions)
- Random Forest ML Model (100% accuracy)
- Personalized recommendations
- 6-panel visualization dashboard
- Confidential report generation
- Screening history tracking
- Crisis helpline information

## Sample Output
- Depression: 8/27 (Mild)
- Anxiety: 4/21 (None/Minimal)
- Stress: 18/40 (Moderate)
- ML Assessment: Mild Issues

## Tech Stack
Python 3.13 | Scikit-learn | Matplotlib | NumPy

## Usage
python mental_health_screener.py
